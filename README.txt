# FlirtBot - שרת ובוט עזר לשיחות עם נשים

## קבצים כלולים:
- server.js — קוד שרת Express שמתחבר ל-OpenAI
- package.json — הגדרות פרויקט
- index.html — דמו גרפי להפעלה מהדפדפן
- README.txt — הוראות

## הפעלה מקומית:
1. צור קובץ `.env` עם השורה:
   OPENAI_API_KEY=sk-XXX

2. התקן תלויות:
   npm install

3. הרץ את השרת:
   npm start

4. פתח את index.html בדפדפן

## פריסה ל-Render:
- העלה את כל הקבצים ל-GitHub
- צור Web Service חדש ב-Render
- השתמש ב-build command: `npm install`
- השתמש ב-start command: `node server.js`
- הוסף משתנה סביבה OPENAI_API_KEY

בהצלחה! 🚀
